# JS Playground Pro

A professional, modern, and production-ready online JavaScript code runner and playground.

## Features

- **Professional Editor**: Powered by Monaco Editor (VS Code engine).
- **Multi-Tab Support**: Edit HTML, CSS, and JavaScript simultaneously.
- **Live Preview**: Real-time iframe rendering of your code.
- **Smart Console**: Captures `console.log`, `warn`, and `error` with timestamps.
- **Glassmorphism UI**: Modern SaaS dashboard design with smooth animations.
- **Resizable Layout**: Flexible panels for a personalized workspace.
- **Keyboard Shortcuts**: `Ctrl + Enter` to run code instantly.
- **Shareable Snippets**: Generate unique URLs to share your code.
- **Auto-Save**: Your progress is automatically saved to local storage.
- **Export/Import**: Download your code as files or copy to clipboard.

## Tech Stack

- **Frontend**: React + Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **State Management**: Zustand
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Editor**: Monaco Editor

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:3000`.
